Mis on HTML?

HTML tähendab HyperText Markup Language – see on veebilehtede põhikeel, mida kasutatakse sisu struktuuri loomiseks. Lihtsalt öeldes: HTML kirjeldab, mis lehel on – pealkirjad, lõigud, pildid, lingid, tabelid, vormid jne.Ilma HTML-ita poleks veebilehti sellisel kujul, nagu me neid näeme.

Mis on CSS?

CSS tähendab Cascading Style Sheets – see on stiililehtede keel, mida kasutatakse HTML-ile välimuse andmiseks. Kui HTML kirjeldab mis on lehel (pealkiri, tekst, pilt, link), siis CSS määrab kuidas see välja näeb.
